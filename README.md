# 我的待办清单 (My To-Do List)

一个简洁的待办事项管理网页应用，用于记录和管理日常任务。

## 在线体验

🔗 https://yanqincao-sys.github.io/to-do-list/

## 功能

- 添加待办任务，支持按 Enter 键快速添加
- 勾选标记任务完成状态
- 删除任务
- 按"全部 / 进行中 / 已完成"筛选查看
- 数据自动保存在浏览器本地，刷新页面不丢失
- 支持自定义主题色（预设色板 + 自由取色）
- 适配手机端浏览，支持添加到主屏幕使用

## 技术栈

原生 HTML / CSS / JavaScript，无第三方框架，使用浏览器 `localStorage` 实现数据持久化。

## 本地运行

下载或克隆本仓库后，直接用浏览器打开 `index.html` 即可使用，无需安装任何依赖。
